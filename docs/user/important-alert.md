# Плашка важного уведомления

`Phoenix:ImportantAlert` — Twig-компонент, который отображает закреплённую сверху плашку для критических сообщений.

```twig
{% set notifications = user_notifications(app.user) %}
<twig:Phoenix:ImportantAlert :message="notifications|first" />
```

- `user_notifications(app.user)` возвращает массив важных сообщений для текущего пользователя.
- `notifications|first` берёт первый элемент списка и передаёт его в свойство `message` компонента.
- Если `message` не пустое, компонент выводит Bootstrap-`alert`, фиксированный в верхней части страницы.
  Скрипт компонента вычисляет высоту плашки после загрузки страницы и при изменении размера окна,
  сдвигая содержимое страницы вниз на соответствующую величину.
- Если сообщения нет, компонент ничего не выводит.
