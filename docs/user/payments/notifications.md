# Уведомления о платежах

После каждого успешного списания система отправляет уведомления, чтобы пользователь сразу знал о новом платеже.

Функция использует компонент [Symfony Notifier](https://symfony.com/doc/current/notifier.html), который объединяет почтовые и SMS‑каналы и позволяет добавлять другие способы доставки (например, чат или мессенджеры).

## Настройка

1. Укажите DSN почтового транспорта в `config/packages/mailer.yaml` через переменную окружения `MAILER_DSN`.
2. Добавьте SMS‑транспорты в `config/packages/notifier.yaml` в секции `framework.notifier.texter_transports`.
3. При необходимости настройте дополнительные каналы в `framework.notifier.channel_transports` и определите правила отправки через `framework.notifier.channel_policy` и получателей по умолчанию.

После этого каждое завершённое списание будет приводить к отправке email и SMS с суммой, датой и информацией о подписке, а другие каналы будут задействованы согласно заданной политике.
