# Регистрация по приглашениям

Параметр `app.user.invite_only` управляет необходимостью ввода действительного кода приглашения при регистрации.

- **По умолчанию:** `true` — регистрация требует действительного приглашения.
- **Если установить в `false`:** пользователи могут зарегистрироваться без кода приглашения. `SignUpController` и `RegisterCommandHandler` учитывают этот флаг.

Установите параметр в `config/packages/app.yaml`:

```yaml
parameters:
    app.user.invite_only: false
```
