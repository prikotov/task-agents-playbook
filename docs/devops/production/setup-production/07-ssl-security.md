# 06. SSL и безопасность

В этом разделе — только автопродление сертификатов и проверка. Выпуск сертификатов выполняйте в разделах для конкретных доменов (03, 04, 05). Certbot устанавливается на этапе подготовки (см. «01. Подготовка production-server»).

## Включение автопродления (systemd timer)
```bash
sudo systemctl enable --now certbot-renew.timer
```

## Проверка автопродления
```bash
sudo certbot renew --dry-run
systemctl is-enabled certbot-renew.timer
systemctl status certbot-renew.timer --no-pager
```

Примечания:
- Certbot пытается продлить сертификаты примерно за 30 дней до истечения.
- После успешного продления перезагрузка nginx обычно не требуется с плагином `--nginx`. Если используете собственные хуки — убедитесь, что они корректно отрабатывают.
