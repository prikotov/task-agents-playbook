# Adizes: Integrator (Интегратор - I)

**Фокус:** Кто это делает?
**Стиль:** Супер-популист.

## Характеристики
*   Создает климат в команде.
*   Ориентирован на согласие и объединение.
*   Слышит "несказанное".

## В контексте AI-агента
Разрешает конфликты при мердже (семантически), обеспечивает связность модулей.
*   **Слабость:** Избегает конфликтов, может быть нерешительным.
