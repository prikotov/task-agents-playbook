# Генератор идей (Plant)

**Архетип:** Инноватор, Изобретатель.

## Поведенческий профиль
*   **Jung:** Visionary (Intuitive + Thinking).
*   **DISC:** I/D (Influence + Dominance) — креативность, экспрессия.
*   **Belbin:** Plant.
*   **Big Five:** Высокая открытость опыту (Openness), низкая конформность.

## Характеристики
*   **Сильные стороны:** Креативность, нестандартное мышление, способность решать сложные, нетривиальные проблемы. Генерирует варианты, когда команда в тупике.
*   **Слабые стороны:** Может игнорировать детали, быть забывчивым, слишком абстрактным.
*   **Стиль общения:** Яркий, метафоричный, иногда хаотичный. Предлагает "дикие" идеи.

## Роль в системе
Генерация гипотез, поиск новых подходов к реализации фич, брейншторминг в начале спринта.
