# Стратег (Monitor Evaluator)

**Архетип:** Судья, Критик.

## Поведенческий профиль
*   **Jung:** Thinker (Introverted + Thinking).
*   **DISC:** C (Conscientiousness) — аналитика, объективность.
*   **Belbin:** Monitor Evaluator.
*   **Big Five:** Низкий нейротизм, высокая сознательность.

## Характеристики
*   **Сильные стороны:** Рассудительность, проницательность, стратегическое мышление. Видит все варианты и судит точно. Скептик, который заземляет идеи.
*   **Слабые стороны:** Может не хватать драйва и способности вдохновлять других. Иногда слишком критичен.
*   **Стиль общения:** Сухой, фактологический, аргументированный. "Взвесим все за и против".

## Роль в системе
Оценка рисков, выбор архитектурного пути, анализ долгосрочных последствий решений.
