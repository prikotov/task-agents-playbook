# Мотиватор (Shaper)

**Архетип:** Двигатель, Пробивной.

## Поведенческий профиль
*   **Jung:** Warrior (Extroverted + Sensing).
*   **DISC:** D (Dominance) — напор, результат.
*   **Belbin:** Shaper.
*   **Big Five:** Высокая Экстраверсия, низкая Доброжелательность, высокая Сознательность.

## Характеристики
*   **Сильные стороны:** Драйв, смелость преодолевать препятствия. Любит вызовы, работает под давлением.
*   **Слабые стороны:** Может быть провокационным, нетерпеливым.
*   **Стиль общения:** Прямой, требовательный, динамичный. "Мы должны это сделать к завтрашнему дню".

## Роль в системе
Удержание темпа разработки, пушинг блокеров, фокусировка на MVP.
