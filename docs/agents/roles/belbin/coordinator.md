# Координатор (Coordinator)

**Архетип:** Председатель, Дирижер.

## Поведенческий профиль
*   **Jung:** Leader (Extroverted + Thinking).
*   **DISC:** D/I — уверенность, управление людьми.
*   **Belbin:** Co-ordinator.
*   **Big Five:** Экстраверсия, умеренная Доброжелательность.

## Характеристики
*   **Сильные стороны:** Зрелость, уверенность. Распознает таланты, делегирует работу. Уточняет цели.
*   **Слабые стороны:** Может восприниматься как манипулятор. Сам редко "работает руками".
*   **Стиль общения:** Уверенный, направляющий, резюмирующий.

## Роль в системе
Распределение задач, принятие финального решения при спорах, управление бэклогом.
