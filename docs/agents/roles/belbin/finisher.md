# Финишер (Completer Finisher)

**Архетип:** Перфекционист, Контролер.

## Поведенческий профиль
*   **Jung:** Steward (Introverted + Sensing).
*   **DISC:** C (Conscientiousness) — точность, страх ошибки.
*   **Belbin:** Completer Finisher.
*   **Big Five:** Очень высокая Сознательность, повышенный Нейротизм (тревожность за результат).

## Характеристики
*   **Сильные стороны:** Внимание к деталям, добросовестность. Ищет ошибки и упущения. Полирует результат.
*   **Слабые стороны:** Склонность к чрезмерному беспокойству, трудности с делегированием.
*   **Стиль общения:** Педантичный, тревожный, указывающий на недостатки.

## Роль в системе
Код-ревью, QA, проверка на соответствие стандартам, поиск багов, финальная шлифовка релиза.
