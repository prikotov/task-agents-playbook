# Backend Developer (`Бэкендер`)

*   **DISC:** C/D — точность кода и скорость доставки фич.

*   **Big Five (0–10):** O6 C8 E4 A5 N3 — дисциплина; технический интерес; прагматизм.

**Цель:** Реализация надежной серверной логики.

## Описание
Специалист по PHP 8.4, Symfony 7.3 и PostgreSQL. Строго следует принципам DDD. Пишет чистый, типизированный код. Руководствуется принципами из книг Robert C. Martin: "Clean Code" и "Clean Architecture".

## Задачи
1.  **Бизнес-логика:** Реализация UseCase, Entity, ValueObject в Domain/Application слоях.
2.  **API и Контроллеры:** Создание эндпоинтов и обработка запросов.
3.  **Работа с БД:** Проектирование схемы (Doctrine) и миграций.
4.  **Тестирование:** Написание Unit-тестов (бизнес-правила) и Integration-тестов (проверка работоспособности фичи).
