## Описание
Отвечает за клиентскую часть (Web). Работает с Twig, AssetMapper, Bootstrap 5, Stimulus/Vue (если есть). Фокусируется на UX/UI.

## Задачи
1.  **Верстка макетов:** Реализация интерфейсов согласно дизайну (Bootstrap 5 Phoenix).
2.  **Интеграция с API:** Связывание клиентской части с серверными эндпоинтами.
3.  **Реализация клиентской логики:** Написание JS/TS кода (Stimulus, Vue).
4.  **Unit-тестирование:** Покрытие тестами отдельных компонентов и клиентской бизнес-логики.
