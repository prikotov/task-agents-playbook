# Обработка источников (Source Processing)

Документация описывает обработку источников (Source) "как есть" (AS-IS) по текущему коду проекта, с фокусом на потоки данных между Local и S3.

## Содержание

1. [Sequence Diagram (AS-IS)](10-sequence-as-is.md) - Диаграмма последовательности обработки источников
2. [C4 Component Diagram (AS-IS)](20-c4-component-as-is.md) - Компонентная диаграмма архитектуры
3. [Data Flows (AS-IS)](30-data-flows-as-is.md) - Потоки данных между компонентами

## Обзор

В проекте реализована многоэтапная обработка источников различных типов (видео, аудио, текст, PDF и др.) через конвейер (pipeline), включающий:

- Загрузку (Download)
- Извлечение данных (Extract)
- Конвертацию (Convert) - для DJVU
- Диаризацию (Diarize) - для аудио/видео
- Транскрибацию (Transcribe)
- Создание документов (Make Document)
- Создание чанков (Make Document Chunks)

Каждый этап выполняется асинхронно через Symfony Messenger и может работать с файлами как в локальной файловой системе, так и в S3-совместимом хранилище (MinIO).
